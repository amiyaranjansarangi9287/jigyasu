# Jigyasu Ambassador Program Guide

## Overview

The Jigyasu Ambassador Program empowers teachers, NGOs, and schools to bring free, visual, multilingual learning to their communities. This guide provides resources and templates for promoting Jigyasu effectively.

## Mission Alignment

Jigyasu exists to:
- **Equity**: One platform. For everyone. The child in a village in Odisha and the child in an apartment in Bengaluru should see the same beautiful concept.
- **Wonder**: We begin with questions, not answers.
- **Identity**: Built for India, from India.

### For Schools

## Getting Started

### 1. Platform Introduction
Jigyasu is a free, offline-first educational platform that:
- Works on 2G connections
- Supports 6 Indian languages (English, Hindi, Tamil, Telugu, Kannada, Odia)
- Requires no installation - just open in browser
- Tracks learning progress locally (privacy-first)
- Uses Wonder-First methodology (curiosity-driven learning)

### 2. Quick Setup for Your Context

**For Classroom Use:**
1. Open jigyasu.app in browser
2. Create learner profiles (one per student or shared device)
3. Select appropriate age group and language
4. Choose modules aligned with curriculum
5. Let students explore at their own pace

**For Community Centers:**
1. Set up a shared device with Jigyasu bookmarked
2. Create a simple sign-in system (name + avatar)
3. Guide learners to appropriate modules
4. Track completion manually if needed

**For School Labs:**
1. Install Jigyasu shortcut on all computers
2. Create class accounts
3. Assign specific modules per grade/subject
4. Use progress data for assessment

## Common Questions

**Q: Is Jigyasu really free?**
A: Yes, forever. No premium versions, no locked content. This is not a feature, it's a principle.

**Q: Does it work offline?**
A: Yes, after initial load. Designed for 2G connections and areas with poor connectivity.

**Q: What about student data?**
A: All data is stored locally on the device. We don't collect or transmit personal information.

**Q: Can multiple students share one device?**
A: Yes, with shared phone mode. Each student creates their own profile.

**Q: How do I know which modules to use?**
A: Modules are organized by age group and subject. Start with topics aligned to your curriculum.

**Q: Can I suggest new topics?**
A: Yes! Email content@jigyasu.app with suggestions. We prioritize topics based on community need.
